Explore your keyboard Your keyboard is a tool that can help you control Narrator. Narrator commands are often abbreviated like this: Narrator key + 1. This means press and hold the Narrator key, then press the number 1 key.?This command turns on input learning, which tells you what keys you’re pressing, as well as the Narrator command associated with them. To turn it off, press Narrator key + 1 twice. You can use Caps lock or Insert as your Narrator key.
